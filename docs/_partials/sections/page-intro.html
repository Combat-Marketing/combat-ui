Page intro

Use cui-page-intro as a page header strip: eyebrow, display title, lead copy, optional meta line, action row, and an optional aside column. Variants tune scale and tone for landing, case, vacancy, and campaign pages.

API reference
Modifier Effect
variant="landing" Display-scale title, wider content column
variant="case" Muted background, two-column with aside at ≥ 56rem
variant="vacancy" Two-column with aside; tighter content measure
variant="campaign" Muted background, two-column with aside
inverse Inverse surface and foreground tokens
full-bleed Span the full inline-size of the parent
align="center" Center-align title, copy, and actions
Slot Purpose
eyebrow Kicker text rendered above the title.
title Display heading for the page.
meta Meta strip — typically a row of <span> labels (sector, year, stack).
(default) Lead copy and paragraphs that follow the title.
actions Primary and secondary CTA buttons.
aside Optional sidebar column, shown to the right of the content at ≥ 56rem. Used by the case, vacancy, and campaign variants.
Variable Default
--cui-page-intro-padding-block --cui-section-space-compact
--cui-page-intro-gap --cui-space-5
--cui-page-intro-content-size 52rem
--cui-page-intro-aside-size 20rem
--cui-page-intro-title-size --cui-font-size-5
--cui-page-intro-bg · --cui-page-intro-color Surface and text overrides.
--cui-page-intro-inverse-bg · --cui-page-intro-inverse-color · --cui-page-intro-inverse-muted-color · --cui-page-intro-inverse-eyebrow-color · --cui-page-intro-inverse-aside-bg Inverse-mode overrides (applied when the inverse attribute is present).
