API reference
Slot Purpose
eyebrow Small kicker text rendered above the title.
title Display heading. Sized via --cui-hero-title-size.
(default) Lead copy and supporting paragraphs.
actions CTA button group. Wrap in a .cui-hero-actions cluster.
media Image, video, or markup for the split / overlay variants. Use the .cui-hero-media class for cover fit.
Attribute / property Purpose
variant="split | background | overlay | text" Layout preset. split shows media beside copy; background renders background-src behind the content; overlay layers content on top of a media slot; text is copy-only.
background-src Image URL used by the background variant. Reflected to the --cui-hero-background-image custom property.
background-position CSS background-position for the background variant. Defaults to center.
background-size CSS background-size for the background variant. Defaults to cover.
align="start | center" Horizontal alignment of the content column.
content-width="narrow | default | wide" Caps the inner content's inline-size to a site container scale while the hero surface stays full-bleed. Omit for the wide default (96rem); use default to align with the rest of the site content (72rem), or narrow for editorial copy (48rem). Drives --cui-hero-container-size.
